# processo seletivo guerra consórcios

## variáveis de ambiente

preencher dados no arquivo .env de acordo com .env.example

## instalar dependências

```bash
pnpm i
```
```bash
npm i
```

## criar banco de dados local

```bash
pnpm db:push
```
```bash
npm run db:push
```

## rodar servidor local

```bash
pnpm dev
```
ou
```bash
npm run dev
```
